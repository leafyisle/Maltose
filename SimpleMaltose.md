# 简易麦芽糖制作过程

## 步骤-1：碾碎
| 项目 | 说明 |
| :--- | :--- |
| **材料** | 基础啤酒大麦芽 |
| **工艺** | 碾碎，磨碎 |
| **工具** | 擀面杖 / 圆形酒瓶 / 磨粉机 |
| **备注** | 待定 |

---

## 步骤-2：糖化
| 项目 | 说明 |
| :--- | :--- |
| **材料** | 碾碎或者磨碎的基础啤酒大麦芽；水 |
| **工艺** | 1. 将材料与水混合，比例约 **1 : 3.5**<br>2. 放入电饭煲中加热至 **65~68℃**，保温 **1 小时** 左右 |
| **工具** | 锅和热源（推荐电饭煲）、温度计、搅拌棒 |
| **备注** | 期间需监测温度并搅拌，推荐频率 **5~10 分钟/次** |

---

## 步骤-3：过滤
| 项目 | 说明 |
| :--- | :--- |
| **材料** | 步骤-2 中的糖化混合物 |
| **工艺** | 将混合物用纱布过滤出液体，倒入蒸发锅中 |
| **工具** | 纱布、蒸发锅 |
| **备注** | 可先用大漏勺粗滤，再用纱布精滤；纱布可用“豆腐布”（搜索词：豆腐布） |

---

## 步骤-4：蒸发
| 项目 | 说明 |
| :--- | :--- |
| **材料** | 步骤-3 中过滤出的液体 |
| **工艺** | 1. 中火加热蒸发锅至煮沸<br>2. 转小火蒸发至液体粘稠 |
| **工具** | 蒸发锅 |
| **备注** | 后期需不停搅拌，避免烧焦 |

> 说明：方法源于网上，我只是搬运。
